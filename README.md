# ptz-utils
ptz camera control utils   





## 1 实现功能  
1.1 打开双目摄像机，启动跟踪，获取视频画面和跟踪信息。   
1.2 打开云台摄像机，手动控制云台的左右上下转动和放大缩小变倍，并获取PTZ值。   
1.3 能够对云台摄像机的PTZ值和跟踪信息进行标定，支持球坐标系、教师坐标系等。并把标定信息保存到配置文件中。   
1.4 对当前的跟踪状态进行导播判断。单目标？还是多目标？     
1.5 基于PID单目标的云台跟踪拍摄。
